原生HTML特性下换肤



link元素,三种不同性质的css样式文件加载



  • 没有title ref 仅仅是 stylesheet 无论如何都会加载 e.g. idnex.css
  • 有title属性 ref 仅仅为 stylesheel 作为默认样式css文件加载并渲染 e.g. default.css
  • 有title属性 rel同时包含alertnate stylesheel 作为备选样式css文件加载,默认不渲染 e.g. red.css && green.css


根据 rel="stylesheel"的link如果有title属性 并且有值 ,性质上就变成了一个可以控制其渲染或者不渲染的特殊元素了



// 渲染red.css document.querySelector('link[href="red.css"]').disabled = false 在把disabled属性设置为false前 要先设置为true
